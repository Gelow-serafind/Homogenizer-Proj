# 接线说明 — HK32F 主控与屏幕、旋钮、电机

本说明基于 `SCH_匀浆机FSH2A_V01_260116_2026-04-04.pdf` 原理图，重点关注 HK32F0301MF4P7C 主控芯片与屏幕、旋钮、电机的连接。

## 1. 总体电源关系
- MCU 和屏幕驱动电源使用 `VCC_5V`。
- 电机驱动电源使用 `VCC_24V`。
- 旋钮开关部分通过 5V 上拉电阻连接到 `VCC_5V`，开关公共端接 `GND`。

---

## 2. 屏幕（LCD/显示驱动）
- 屏幕显示驱动芯片为 `TM1729`。
- MCU 与显示驱动之间采用两线串行接口：
  - `MCU_LCD_SDA`
  - `MCU_LCD_SCL`
- 这两条信号线直接从 HK32F0301MF4P7C 引出到 TM1729。
- TM1729 负责驱动 LCD 模组的 `COM0`~`COM3` 和 `SEG31`~`SEG39` 等段线。
- 显示模块本身的背光电源由 `VCC_5V` 提供，通常通过专用 LED 背光引脚连接。

### 主要连接点
- MCU -> TM1729: `MCU_LCD_SDA`、`MCU_LCD_SCL`
- 5V 电源：`VCC_5V`
- 显示模组公共地：`GND`

### 与老项目一致的 MCU 引脚映射（用于代码移植）
- `MCU_LCD_SCL` -> `PC6`
- `MCU_LCD_SDA` -> `PC7`
- 通信方式：GPIO 模拟两线串行（时序与老项目 `tm1729_init1()/gtm1729_disp()` 兼容）

### 当前阶段屏显 Demo 说明
- 已在当前项目 `src/main.c` 内联移植 TM1729 基础驱动（不拆分文件）。
- 上电后会先显示固定测试数字，随后按状态机显示 4 位演示码：
  - `IDLE` -> `1111`
  - `SETTING` -> `2222`
  - `RUNNING` -> `3333`
  - `PAUSED` -> `4444`
- 该 Demo 目标是先确认屏幕点亮与刷屏链路正常，后续可再替换为业务字段显示。

> 注意：原理图中并未显示 MCU 直接驱动 LCD 段线，LCD 段线由 TM1729 驱动芯片控制。

## 3. 旋钮（编码器按钮）
- 旋钮连接器为 `CN1`，类型为 `ZX-XH2.54-4PZZ`。
- 旋钮信号由三个开关/按键组成：
  - `MCU_SW_A`
  - `MCU_SW_B`
  - `MCU_SW_PUSH`
- 这三路信号均通过 10kΩ 上拉电阻连接到 `VCC_5V`。
- 旋钮公共地接 `GND`。

### 旋钮连接器 `CN1` 引脚对应
1. `MCU_SW_A`
2. `MCU_SW_B`
3. `MCU_SW_PUSH`
4. `GND`

### MCU 引脚映射
- `MCU_SW_A` -> HK32F0301MF4P7C 上的 `PC3`
- `MCU_SW_B` -> HK32F0301MF4P7C 上的 `PC4/AIN2/WKUP1`
- `MCU_SW_PUSH` -> HK32F0301MF4P7C 上的 `PC5/EXTCLK4`

---

## 4. 电机
- 电机连接器为 `U5`，类型为 `2EDG-3.81-2A`。
- 电机使用 24V 电源：`VCC_24V`。
- 电机控制信号为 `MCU_2MOTOR_PWM`，由 HK32F0301MF4P7C 输出。

### MCU 与电机控制关系
- `MCU_2MOTOR_PWM` 接到 MCU 的 `PD2/AIN4`。
- 该信号进入内置电机驱动电路，通过 MOSFET `Q2` 驱动外部电机。

### 电机连接器 `U5` 引脚对应
1. `VCC_24V`
2. 电机负极/控制输出，连接到电机驱动回路

### 其它保护/驱动元件
- `D1` 为续流二极管，用于电机断电时吸收反向感应电压。
- `Q2` 为功率 MOSFET，从 MCU 的 PWM 信号控制电机启停。

---

## 5. 串口调试接口
- 调试串口连接器为 `H1`，型号 `PZ254V-11-03P`。
- MCU 通过 `USART1` 提供调试串口信号：
  - `USART1_TX`
  - `USART1_RX`
- `H1` 共 3 针：
  1. `USART1_TX`（MCU 发到调试器/串口适配器）
  2. `USART1_RX`（MCU 收自调试器/串口适配器）
  3. `GND`
- 该调试接口用于串口调试输出和命令交互，通常可连接 USB 转 TTL 适配器或外部串口调试模块。

### H1 调试口说明
- `H1` 引脚 1：`USART1_TX`
- `H1` 引脚 2：`USART1_RX`
- `H1` 引脚 3：`GND`

---

## 6. 关键结论
- 屏幕接口：`MCU_LCD_SDA` + `MCU_LCD_SCL`，通过 TM1729 驱动 LCD。
- 旋钮接口：`MCU_SW_A`、`MCU_SW_B`、`MCU_SW_PUSH` + `GND`，通过 4P 连接器接入。
- 电机接口：`VCC_24V` + 电机输出回路，受 `MCU_2MOTOR_PWM` 控制。
